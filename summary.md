### Отчёт по итогам автоматизации

Был запланирован следующий перечень автоматизируемых сценариев:
1. Корректное отображение формы при выборе типа покупки
2. Ввод валидных данных в поля формы
3. Ввод невалидных данных в поля формы
4. Очистка полей формы
5. Отправка валидных данных
6. Отправка невалидных данных

С использованием следующих инструментов:
1. Docker
2. JUnit5
3. Gradle
4. Selenide
5. Java

Вышеперечисленные сценарии были в полном объёме автоматизированы с использованием всех заявленных инструментов.

По причине неопытности процесс редактирования авто-тестов проводился параллельно с составлением отчётности, 
но глобально на расхождение запланированного и потраченного времени это не повлияло. 
Процессы написания, прогона авто-тестов, а так же составления отчётности были осуществлены за указанные периоды 
(без учёта проверки преподавателем):
- Авто-тесты - 7 дней
- Отчётные документы по итогам тестирования - 3 дня
- Отчёт по автоматизации - 4 дня
